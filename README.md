# Installing

How to run this project:

- Create a folder and navigate into it

```
cd <folder-name>
```

- Clone this repository:

```
git clone https://github.com/mariav53/react-hooks-movie-app
```

- Install the project's dependencies:

```
yarn or npm install
```

- run the project:

```
npm start or yarn start
```
